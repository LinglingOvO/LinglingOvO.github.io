Markdown 语法学习

CoolPine2025-02-12

Description：本文是对个人的 Markdown 语法知识进行复习或补充。有关 Markdown 语法的具体内容，可以查看参考资料中的文档。

1 基础语法

1.1 斜体

  • 语法：*斜体*
  • 示例输出：斜体

1.2 粗斜体

  • 语法：***粗斜体***
  • 示例输出：粗斜体

1.3 引用

  • 语法：
  • 示例输出：

    引用语法

    引用语法

    引用语法

    引用语法

1.4 高亮

  • 语法：==想要高亮的内容==
  • 示例输出：==想要高亮的内容==

1.5 删除线

  • 语法：~~想要设置删除线的内容~~
  • 示例输出：想要设置删除线的内容

1.6 下划线

  • HTML 语法：<u>想要设置下划线的内容</u>
  • 示例输出：想要设置下划线的内容

1.7 编写行内代码

如果要编写行内代码或在短语中包含一个或多个反引号，可以用双反引号 (``) 将代码或短语括起来，也可以使用 HTML 语法。

  • 语法：
    1. Markdown 形式：``在你的 Markdown 文件中使用 `code`。``
    2. HTML 形式：<code>在你的 Markdown 文件中使用 `code`。</code>
  • 示例输出：在你的 Markdown 文件中使用  code。

1.8 上下标

  • HTML 语法：H<sub>2</sub>O，X<sup>2</sup>
  • 示例输出：H2O，X2

注意：在 Obsidian 中 Markdown 的上下标语法可能不生效。

1.9 使用表情符号

通过键入表情符号简码来插入表情符号。这些简码以冒号 (:) 开头和结尾，并包含表情符号的名称。

  • 语法：:joy:
  • 示例输出：😂

    注意：通常可以使用此表情符号简码列表，但表情符号简码可能因应用程序而异。有关更多信息，可以参阅所使用的 Markdown 应用程序文档。

1.10 Markdown 中的 HTML

1.10.1 居中

  • 使用 HTML 中的标签 <center>。
  • 语法：<center>This text is centered.</center>
  • 示例输出：
This text is centered.

1.10.2 更改文本颜色

  • Markdown 语法不支持更改文本颜色，但可以使用 HTML 中的标签  <font>，通过修改其中的 color 属性值，即可更改文本颜色。
  • 可以使用颜色的名称或十六进制  #RRGGBB  代码修改 color 属性的值。
  • 语法：<font color="red">This text is red!</font>
  • 示例输出：This text is red!

1.10.3 调整图像大小

  • 使用 HTML 中的标签 img，通过修改其 width 和 height 属性来以像素为单位设置图像的尺寸。
  • 语法：<img src="https://111-1321365386.cos.ap-beijing.myqcloud.com/20250213040901088.jpg" width="200" height="100">
  • 示例输出：
  • 原图：

1.10.4 符号

Markdown 没有为符号提供特殊语法。若想使用符号，有 2 种方式：

  1. 在大多数情况下，可以将任何你想要使用的符号复制并粘贴到 Markdown 文档中。
  2. 可以使用 HTML 实体来表示任何你想要使用的符号。符号的 HTML 实体的部分列表如下：
    • 版权 (©) — &copy;
    • 注册商标 (®) — &reg;
    • 商标 (™) — &trade;
    • 欧元 (€) — &euro;
    • 左箭头 (←) — &larr;
    • 上箭头 (↑) — &uarr;
    • 右箭头 (→) — &rarr;
    • 下箭头 (↓) — &darr;
    • 度数 (°) — &#176;
    • 圆周率 (π) — &#960;

2 使用技巧

2.1 标题：

在文档创作中，一般采用如下的标题层级结构：

  1. # 文章主标题 （一级标题，对应 Front Matter 中的  title  字段）
  2. ## 核心章节 （二级标题，用于主体内容划分）
  3. ### 子章节 （三级标题，用于细节内容展开）
  4. #### 细分段落 （四级标题，建议不超过此层级）

注意：

  • Markdown 语法最多支持 6 级标题
  • 注意文章的层级连续性，避免出现跳级使用（如一级标题后直接使用三级标题）

2.2 列表

Markdown 语法中，有序列表和无序列表之间可以互相嵌套使用。

2.3 链接

可以选择为链接添加提示。当用户将鼠标悬停在链接上时，即可显示提示。

  • 语法：[Google Search](https://www.google.com.hk/ "谷歌搜索")
  • 示例输出：Google Search

若想在图片中添加链接，可以用方括号括住图片的 Markdown，然后在括号中添加链接，即嵌套使用。

  • 语法：[![MyImage](https://111-1321365386.cos.ap-beijing.myqcloud.com/20250213040901088.jpg)](https://www.bing.com/search?q=%E7%8B%97)
  • 示例输出：

2.4 缩进

  • HTML 语法：
    • &nbsp;&nbsp;&nbsp;&nbsp;段落的第二句。
    • 此处的&nbsp;相当于空格。
  • 示例输出：
    •     段落的第二句。
    • 此处的 相当于空格。

注意：在编写 Markdown 文档时，除非段落位于列表中，否则不要使用空格或制表符缩进段落。
